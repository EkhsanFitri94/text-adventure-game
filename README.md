Text Adventure Game
A simple text-based adventure game created as my first project while learning Python game development.

How to Run
1. Make sure you have Python installed.
2. Open a terminal or command prompt.
3. Navigate to the directory containing game.py.
4. Run the command: python game.py

How to Play
- Type commands like right, ahead, or back to move between rooms.
- The goal is to find the treasure room!

Skills Demonstrated
- Python variables and data types (dictionaries, strings, booleans)
- Control flow (if/elif/else statements)
- Functions
- Basic game loop and state management
